# 📖 TOEFL Vocab Trainer

플래시카드 · 퀴즈 · 단어장 3가지 모드로 공부하는 **TOEFL 빈출 어휘** 학습 웹앱입니다.
설치·서버 없이 `index.html`만 열면 바로 동작하며, 학습 진행 상황은 브라우저에 자동 저장됩니다.

## ✨ 기능

- 🃏 **플래시카드** — 카드를 뒤집어 뜻/예문 확인, 발음 듣기(TTS), 외움/즐겨찾기 표시, 셔플
- ✍️ **퀴즈** — 4지선다 10문항, 즉시 채점 및 점수 결과
- 📚 **단어장** — 전체 목록, 검색(영어·한글), 외움/즐겨찾기 필터
- 📊 **진행률 추적** — 외운 단어 수/퍼센트 표시, `localStorage` 자동 저장
- 🌙 다크 모드 · 📱 반응형 · ⌨️ 키보드 단축키

### 키보드 단축키 (플래시카드)
| 키 | 동작 |
|----|------|
| `Space` | 카드 뒤집기 |
| `←` `→` | 이전 / 다음 단어 |
| `Enter` | 외움 표시 후 다음으로 |

## 🚀 사용법

- **바로 실행:** `index.html` 더블클릭 (별도 서버 불필요 — 단어 데이터가 내장돼 있음)
- **웹 배포:** GitHub Pages 등 정적 호스팅에 올리면 끝

## 📂 구성

```
toefl-vocab/
├── index.html   # 앱 (UI + 로직)
└── words.js     # 단어 데이터 (102개)
```

## 📝 단어 추가/수정

`words.js`의 배열에 항목을 추가하면 됩니다.

```js
{ word: "abandon", pos: "v.", en: "to leave completely", ko: "버리다, 포기하다", ex: "The crew had to abandon the ship." }
```

| 필드 | 설명 |
|------|------|
| `word` | 영어 단어 |
| `pos` | 품사 (v. / n. / adj. 등) |
| `en` | 영영 뜻 |
| `ko` | 한글 뜻 |
| `ex` | 예문 |
