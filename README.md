# 📖 TOEFL Vocab & Speaking Trainer

**난이도별 TOEFL 어휘 500개** + **오픽(OPIc)·토플 스피킹 예시답변**으로 공부하는 영어 학습 웹앱입니다.
설치·서버 없이 `index.html`만 열면 바로 동작하며, 학습 진행 상황은 브라우저에 자동 저장됩니다.

🔗 **라이브:** https://slow0704-eng.github.io/toefl-vocab/

## ✨ 기능

### 어휘 (500단어 · 난이도 3단계)
- 🟢 **기초** · 🟠 **중급** · 🔴 **고급** 으로 분류, 난이도별 필터 학습
- 🃏 **플래시카드** — 카드 뒤집기, 발음 듣기(TTS), 외움/즐겨찾기, 셔플
- ✍️ **퀴즈** — 4지선다 10문항(난이도 선택 가능), 즉시 채점
- 📚 **단어장** — 검색, 외움/즐겨찾기/난이도 필터
- 📊 진행률 자동 저장(`localStorage`)

### 말하기 (오픽 · 토플 스피킹 예시답변)
- 🗣 **OPIc** 20개 주제 — 자기소개·집·가족·여가·여행·롤플레이 등, IH~AL 수준 모범답변
- 🎓 **TOEFL Speaking** 9개 — 독립형(Task 1) 모범답변 6개 + 통합형(Task 2·3·4) 답변 템플릿
- 각 주제마다: **영문 답변** · **한글 번역 토글** · **원어민 음성 재생(TTS)** · **핵심 표현** · **고득점 팁**
- 시험·주제·검색 필터

공통: 🌙 다크 모드 · 📱 반응형 · ⌨️ 키보드 단축키

### 키보드 단축키 (플래시카드)
| 키 | 동작 |
|----|------|
| `Space` | 카드 뒤집기 |
| `←` `→` | 이전 / 다음 단어 |
| `Enter` | 외움 표시 후 다음으로 |

## 🚀 사용법

- **바로 실행:** `index.html` 더블클릭 (서버 불필요 — 데이터가 내장돼 있음)
- **웹 배포:** GitHub Pages 등 정적 호스팅에 올리면 끝

## 📂 구성

```
toefl-vocab/
├── index.html    # 앱 (UI + 로직)
├── words.js      # 어휘 데이터 500개 (난이도 포함)
└── speaking.js   # 오픽·토플 스피킹 예시답변 29개
```

## 📝 데이터 추가/수정

**단어** — `words.js` 배열에 추가 (`level`: 1=기초, 2=중급, 3=고급)
```js
{ word: "mitigate", pos: "v.", en: "to make less severe", ko: "완화하다", ex: "Trees can mitigate heat.", level: 2 }
```

**말하기 주제** — `speaking.js` 배열에 추가
```js
{ exam: "OPIc", category: "여가·취미", question: "...", targetLevel: "IH–AL",
  answerEn: "...", answerKo: "...", keyExpressions: [{en:"...", ko:"..."}], tips: "..." }
```
