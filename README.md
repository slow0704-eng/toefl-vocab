# 📖 영어시험 올인원 — Vocab · Speaking · Writing · Reading Trainer

**난이도별 TOEFL 어휘 500개** + **오픽(OPIc)·토익스피킹·토플 스피킹/라이팅/리딩 문항 206개**로 공부하는 영어 학습 웹앱입니다.
설치·서버 없이 `index.html`만 열면 바로 동작하며, 학습 진행 상황은 브라우저에 자동 저장됩니다.

🔗 **라이브:** https://slow0704-eng.github.io/toefl-vocab/

## ✨ 기능

### 어휘 (500단어 · 난이도 3단계)
- 🟢 **기초** · 🟠 **중급** · 🔴 **고급** 으로 분류, 난이도별 필터 학습
- 🃏 **플래시카드** — 카드 뒤집기, 발음 듣기(TTS), 외움/즐겨찾기, 셔플
- ✍️ **퀴즈** — 4지선다 10문항(난이도 선택 가능), 즉시 채점
- 📚 **단어장** — 검색, 외움/즐겨찾기/난이도 필터
- 📊 진행률 자동 저장(`localStorage`)

### 스피킹 · 라이팅 · 리딩 (206문항 · 140개 상세주제)

| 시험 | 문항 | 구성 |
|------|------|------|
| 🗣 **OPIc** | 138 | 기본 주제 · 여가활동 · 취미/관심사 · 운동 · 휴가/출장 · 돌발주제 · 경험 · 롤플레이 |
| 💼 **토익스피킹** | 17 | Q1-2 지문 읽기 · Q3-4 사진 묘사 · Q5-7 질문에 답하기 · Q8-10 표 보고 답하기 · Q11 의견 제시 |
| 🎓 **토플 스피킹** | 17 | 독립형(Task 1) 6 · 통합형(Task 2·3·4) 3 · **스피킹 템플릿 8** |
| ✍️ **토플 라이팅** | 14 | Integrated Writing 5 · Academic Discussion 7 · 라이팅 템플릿 2 |
| 📖 **토플 리딩** | 20 | 문제 유형 10종 해설 · 빈출 지문 주제 10종 |

- **OPIc은 설문 상세주제 단위**로 정리 — 영화 보기, 공원 가기, 음악 감상, 요리, 반려동물, 헬스, 조깅, 요가, 자전거, 수영, 국내/해외 여행, 날씨, 교통, 은행, 재활용, 호텔 등
- 상세주제마다 **묘사 / 경험(과거) / 변화·비교(또는 문제 상황)** 3콤보 질문과 답변
- 롤플레이 8개 상황 × (정보 요청 + 문제 해결) 2세트
- 각 문항마다: **영문 답변** · **한글 번역 토글** · **원어민 음성 재생(TTS)** · **핵심 표현 5개** · **고득점 팁**
- 시험 → 분류 → 상세주제 3단계 필터 + 통합 검색

공통: 🌙 다크 모드 · 📱 반응형 · ⌨️ 키보드 단축키

### 키보드 단축키 (플래시카드)
| 키 | 동작 |
|----|------|
| `Space` | 카드 뒤집기 |
| `←` `→` | 이전 / 다음 단어 |
| `Enter` | 외움 표시 후 다음으로 |

## 🚀 사용법

- **바로 실행:** `index.html` 더블클릭 (서버 불필요 — 데이터가 내장돼 있음)

```
toefl-vocab/
├── index.html    # 앱 (UI + 로직)
├── words.js      # 어휘 데이터 500개 (난이도 포함)
└── speaking.js   # 스피킹·라이팅·리딩 문항 206개
```

## 📝 데이터 추가/수정

**단어** — `words.js` 배열에 추가 (`level`: 1=기초, 2=중급, 3=고급)
```js
{ word: "mitigate", pos: "v.", en: "to make less severe", ko: "완화하다", ex: "Trees can mitigate heat.", level: 2 }
```

**문항** — `speaking.js` 배열에 추가
(`exam`: `OPIc` | `TOEIC` | `TOEFL` | `TOEFL Writing` | `TOEFL Reading`)
```js
{ exam: "OPIc", category: "여가활동", topic: "영화 보기", question: "...", targetLevel: "IH–AL",
  answerEn: "...", answerKo: "...", keyExpressions: [{en:"...", ko:"..."}], tips: "..." }
```
지문·보기처럼 줄바꿈이 필요한 곳은 `\n`을 쓰면 화면에서 줄바꿈으로 표시됩니다.
